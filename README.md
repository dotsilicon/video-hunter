# 🎬 Spielberg – YouTube Video Downloader

Spielberg is a modern YouTube Video Downloader built with Python and Tkinter.  
It provides a clean UI, live video quality selection, and fast downloads powered by yt-dlp.

---

## ✨ Features

- 📋 Auto-detect YouTube links from clipboard
- 🔗 Auto-fill video URL input box
- 🎥 Live per-video quality list (resolution-based)
- ⏯️ Start / Pause download control
- 📁 Custom download folder selection
- ⚡ Fast downloading using yt-dlp
- 🧠 Smart UI with ttkbootstrap (Solar theme)
